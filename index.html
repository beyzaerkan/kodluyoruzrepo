Git Komutları

Projeye yeni git dizini oluşturur => git init

Projeyi klonlar => git clone projeLinki

Dizine dosyalar ekler => git adddosyaAdı / git add . (tüm dosyaları ekler)

Değişiklikleri yerel depoya gönderir => git commit

Projedeki değişiklikleri uzak depoya gönderir => git push

Uzak depodaki değişiklikleri projeye ekler => git pull

Yeni bir branch oluşturur => git branch branchİsmi

Branchler arası geçiş sağlar => git checkout branchİsmi

Projedeki değişiklikler hakkında bilgi verir => git status

Commit geçmişini görüntüler => git log

Değişikliklerden sonra dosyalar arasında oluşan farklılıkları gösterir => git diff
